⚡ Edge-cache speed demo

Read the same object three ways — raw S3, the Source Cooperative proxy, and the PR #189 chunk-cache preview — straight from your browser, and watch the difference per access pattern.

Cold = first request (cache MISS + populate). Warm = median of the rest. All requests use cache: no-store so the browser cache never hides the network. The direct S3 endpoint reports no x-cache — it has no proxy. Numbers are network-dependent; judge the ratios, and the x-cache disposition, not absolute ms.
